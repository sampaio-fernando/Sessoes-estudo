# Temporizador de Estudo (React Native - Expo)

Aplicativo simples de temporizador de sessões de estudo, construído com React Native (Expo) usando React Hooks.

## Funcionalidades
- Iniciar / Pausar / Resetar o temporizador
- Configurar minutos via TextInput (teclado numérico)
- Mostra tempo em MM:SS
- Muda cor do display quando faltar menos de 60s
- Exibe mensagem quando o tempo termina e vibra (opcional)
- Estatísticas: sessões completas e tempo total estudado

## Rodando localmente
1. Instale o Expo CLI:
   ```bash
   npm install -g expo-cli
## Para rodar o projeto
